Live streams can be published to the server with the following protocols and codecs:

| protocol                                                                | variants                                          |
| ----------------------------------------------------------------------- | ------------------------------------------------- |
| [SRT clients](../3-publish/01-srt-clients.md)                           |                                                   |
| [SRT cameras and servers](../3-publish/02-srt-cameras-and-servers.md)   |                                                   |

| [WebRTC servers](../3-publish/04-webrtc-servers.md)                     | WHEP                                              |
| [RTSP clients](../3-publish/05-rtsp-clients.md)                         | UDP, TCP, RTSPS                                   |
| [RTSP cameras and servers](../3-publish/06-rtsp-cameras-and-servers.md) | UDP, UDP-Multicast, TCP, RTSPS                    |
